# Louis Jefferson — Portfolio

Personal portfolio site for Louis Jefferson — creative direction, content strategy, shoot &amp; edit. Gold Coast, Australia.

Static site (plain HTML/CSS/JS, no build step). Hosted on GitHub Pages at **[louisjefferson.com](https://louisjefferson.com)**.

## Structure
- `index.html` — the entire site
- `assets/logos/` — client logo wall
- `assets/case/` — Ahipara case-study screenshots
- `assets/thumbs/` — work-card video thumbnails
- `assets/gallery/web/` — optimized stills gallery
- `assets/og-image.jpg` — social share card

Camera-original photos in `assets/gallery/` are gitignored; only the optimized `web/` copies are deployed.
